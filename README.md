# Lab 3 - Responsive Portfolio

### Description
One-page responsive portfolio built using Flexbox (header & hero), CSS Grid (skills & projects), media queries, and CSS animations/transitions.

### How to view
1. Open `index.html` in a browser.
2. Test responsive behavior by resizing the browser or using device emulation.

### Breakpoints tested
- Mobile-first layout (<600px)
- Tablet (600px - 899px)
- Desktop (>=900px)
